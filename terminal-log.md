# Terminal Log - Week 03

## Task 5.1: Terminal Navigation

### Commands Used:

1. pwd - showed current directory.
2. ls - listed contents of directory
3. cd ~/Documents - navigated to Documents folder
4. cd .. - went back one directory
5. cd ~ - navigated to home directory

## Task 5.2: Project Structure Creation

6. mkdir src, docs, tests - created main folders
7. mkdir src\css, src\js, src\images - created nested folders
8. New-Item -ItemType File -Path 'README.md' -Force - created README file
9. New-Item -ItemType File -Path 'src\index.html' -Force - created index.html
10. New-Item -ItemType File -Path 'src\css\styles.css' -Force - created styles.css
11. New-Item -ItemType File -Path 'src\js\main.js' -Force - created main.js
